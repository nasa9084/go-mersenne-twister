Implementation of Mersenne Twister in Go

## Usage

``` go
package main

import (
    "fmt"
    "github.com/nasa9084/go-mersenne-twister"
)

func main() {
    mt.InitGenrand(0) // '0' is a seed. choose freely.
    n := mt.GenrandInt32() // generate int 32 random value.
    fmt.Printf("%d", n)
}
```
